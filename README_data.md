Este proyecto utiliza datos públicos procedentes de:

- Instituto Nacional de Estadística (INE)
- Servicio Público de Empleo Estatal (SEPE)

Los archivos originales no se incluyen en el repositorio para:
- reducir peso
- evitar duplicación de datos oficiales
- fomentar la reproducibilidad

Para reproducir el análisis:
1. Descarga los CSV correspondientes a renta (INE) y paro (SEPE).
2. Colócalos en esta carpeta (`data/`).
3. Ajusta las rutas en el script si es necesario.
